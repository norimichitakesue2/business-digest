# business-digest

毎日のビジネスニュース・デイリーダイジェスト（HTML）。

- `index.html` … ダッシュボード（業界の動き2026 / 掘り下げライブラリ / 日次一覧）
- `digest_YYYY-MM-DD.html` … 日次ダイジェスト（連動フロー図・時事・企業分析・今後起きそうなこと）
- `build_digest.py` … day_data.json を受け取りHTMLを生成する決定的レンダラ
- `trends.json` / `deepdives.json` … 業界の動き・掘り下げの永続ストア

GitHub Pages で配信。
