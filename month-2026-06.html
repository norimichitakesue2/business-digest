カバー業界
AI半導体への資金集中（キオクシアが時価総額でトヨタ超え→乱高下）、ウォーシュFRB発足とECB利上げによる金利上昇、中東情勢に揺れる原油、日米関税合意の対米投資5500億ドル——が6月の四大テーマ。詳しくは下の主要ストーリーと各日のダイジェストへ。

26月の主要ストーリー10

「キオクシア・ショック」とAI半導体の一極集中
6月中旬、キオクシアが時価総額60兆円超でトヨタを抜き国内首位に。日経平均は一時7万2000円超の最高値を更新する一方、26日には1889円安と下落幅史上3位の乱高下。AI向けSSD/HBM需要が相場を牽引したが、PER過熱と少数値がさ株への集中が脆さも露呈した。
キオクシア / エヌビディア(Vera Rubin) / 東京エレクトロン
ウォーシュFRB発足と世界的な利上げ転換
新議長ウォーシュの下でFRBはタカ派化し、ドットは『年内利上げ1回』へ転換。ECBは2年9カ月ぶりに利上げ、日銀も年内正常化観測が強まった。長期金利上昇とドル高が高PERのハイテク株の重荷に。
FRB / ECB / 日銀
中東情勢と原油の乱高下
イラン・イスラエルの軍事衝突とホルムズ海峡リスクで原油が急騰した後、停戦観測と供給過剰観測でWTIは68ドル台へ急落。エネルギー・素材・海運・インフレ見通しに大きく波及した。
原油 / ホルムズ海峡 / 海運
対米投資5500億ドルとメガバンクのドル調達
日米関税合意に伴う5500億ドルの対米投融資が始動。JBICが約1/3、3メガが約2/3を担う想定で、全額ドル建ての長期調達が銀行の与信余力を圧迫。政府にドル調達支援を要請する動きに。
三菱UFJ / 三井住友 / みずほ / JBIC
ソフトバンクG×OpenAIとAIインフラ投資
SBGはOpenAI関連の投資益が6.4兆円に達し、一時国内時価総額首位を奪取。Stargate構想やArmと連携しAIインフラへ集中投資。OpenAIのIPO観測も浮上した。
ソフトバンクG / OpenAI / Arm
中国経済の減速と不動産不況
5月の小売売上がマイナス、固定資産投資も減少。保利発展の信託償還延期など不動産発の信用不安が続き、26年は4.5%成長へ減速見通し。7月の政治局会議での追加策に関心。
保利発展 / 万科 / 内需
自動車の出遅れと米関税
トヨタなど自動車株はAI相場・米関税で出遅れ、PBR1倍割れ局面も。HV需要は底堅く、全固体電池・EV・SDVが中期の鍵。円安は輸出採算にプラスに働いた。
トヨタ / ホンダ / BYD
任天堂 Switch 2 の好発進
新型ハードSwitch 2が発売7週で600万台超。4-6月期は売上が前年同期比2.3倍、純利益960億円と急回復。株主総会で『普及を最優先』と表明した。
任天堂
製薬のパテントクリフと業界再編
2026〜30年の大型特許切れ（パテントクリフ）を控え、ジーエヌアイのあゆみ製薬買収などM&Aが活発化。半導体からの資金分散先として医薬ディフェンシブにも物色が向かった。
武田 / 第一三共 / ジーエヌアイ
防衛・宇宙とデジタルインフラ外交
FY2026防衛予算は過去最高で、三菱重工の防衛宇宙が1兆円規模へ拡大。総務省の『デジタル回廊構想』など、経済安全保障を軸にした海底ケーブル・衛星のインフラ外交も進んだ。
三菱重工 / 川崎重工 / IHI
